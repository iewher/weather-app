# Weather app

Weather-app - приложение, показывающее актуальные погодные данные, такие как:

1. Градусы
2. Ветер
3. Влажность
4. Состояние (например Снег)

## Запуск проекта

1. Установить зависимости - `npm install`
1. Запустить проект - `npm run dev`

## Перед запуском

1. Заменить API_KEY в .env файле.
